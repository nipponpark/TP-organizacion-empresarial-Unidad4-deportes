# TP-organizacion-empresarial-Unidad4-deportes
Trabajo práctico - Organización Empresarial - Análisis de resultados deportivos - Git, GitHub y Jira.
El objetivo del proyecto es analizar resultados de partidos utilizando herramientas como Python, Git, GitHub, Jira y Google Colab.

# Integrante
- Ghersinich Nicolás

# Escenario elegido 
Escenario D - Estadísticas de resultados deportivos.

# Descripción del dataset utilizado
Se utilizará un conjunto de datos deportivos que contiene información sobre partidos, equipos y resultados, incluyendo fechas de los encuentros.

# Ejecución del script
- Clonar el repositorio.
- Abrir el proyecto en Google Colab.
- Ejecutar el script en la carpeta /scripts
- Los resultados se guardarán en la carpeta /resultados
